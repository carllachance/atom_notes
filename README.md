# Clarify the Complex (atom_notes)

A minimal Tauri + React + TypeScript prototype for an ambient thinking surface focused on:

- summon/dismiss via double-tap Ctrl
- quick capture via Ctrl+Shift+N
- persistent spatial cards
- anchor + archive lifecycle
- exact workspace state restoration

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
